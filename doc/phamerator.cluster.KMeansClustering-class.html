Class KMeansClustering

source code

Implementation of the kmeans clustering method as explained in http://www.elet.polimi.it/upload/matteucc/Clustering/tutorial_html/kmeans.html

USAGE

>>> from cluster import KMeansClustering
>>> cl = KMeansClustering([(1,1), (2,1), (5,3), ...])
>>> clusters = cl.getclusters(2)
